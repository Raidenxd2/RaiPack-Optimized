# RaiPack-Optimized
An optimized modpack for Minecraft (beta)

# Features
High FPS <br />
Some Optifine features are supported. <br />
Controller support. <br />
You can get 30-50 FPS in a Hypixel Skyblock lobby. <br />
Faster world loading. <br />
Faster game loading. <br />
